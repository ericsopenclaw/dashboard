# 🎧 Focus Flow

**Ambient Timer for Deep Work** — A beautiful, distraction-free focus timer with synthesized ambient sounds.

![Focus Flow](https://img.shields.io/badge/status-ready-brightgreen) ![No Dependencies](https://img.shields.io/badge/dependencies-none-blue) ![PWA Ready](https://img.shields.io/badge/PWA-ready-purple)

## ✨ Features

- **🎯 Pomodoro Timer** — Preset durations (5/25/45/60 min) with visual progress ring
- **🔊 Ambient Sound Mixer** — 8 procedurally-generated ambient sounds (rain, thunder, forest, ocean, fire, café, wind, birds)
- **📊 Focus Stats** — Track sessions, minutes, and daily streaks
- **🔔 Notifications** — Browser notifications when sessions complete
- **⌨️ Keyboard Shortcuts** — Space to start/pause, R to reset
- **📱 Responsive** — Works beautifully on desktop and mobile
- **🌙 Dark Mode** — Easy on the eyes for long focus sessions

## 🚀 Quick Start

Just open `index.html` in a browser — no build step, no dependencies!

```bash
# Or serve locally
npx serve .
```

## 🎵 Sound System

All ambient sounds are **procedurally generated** using the Web Audio API — no external audio files needed!

| Sound | Character |
|-------|-----------|
| 🌧️ Rain | Soft white noise, low-pass filtered |
| ⛈️ Thunder | Deep rumble with low-frequency oscillation |
| 🌲 Forest | Layered nature sounds with bird-like chirps |
| 🌊 Ocean | Rhythmic wave-like modulation |
| 🔥 Fireplace | Crackling warmth |
| ☕ Café | Background chatter ambiance |
| 💨 Wind | Gentle howling breeze |
| 🐦 Birds | High-pitched melodic chirping |

Mix multiple sounds together for your perfect focus environment!

## 🛠️ Tech Stack

- Vanilla HTML/CSS/JavaScript
- Web Audio API for sound synthesis
- CSS custom properties for theming
- LocalStorage for stats persistence
- No frameworks, no build tools, no dependencies
